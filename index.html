About Me

I'm a software developer with experience building responsive, user-focused web applications and interactive digital experiences. My work has involved developing front-end features using modern JavaScript frameworks, contributing to full-stack applications, and collaborating within Agile development teams to deliver high-quality software solutions.

I have experience working across the software development lifecycle, from developing new features and enhancing existing functionality to testing, debugging, and maintaining production applications. I enjoy turning complex requirements into intuitive user experiences and applying strong UI/UX principles to create accessible and reliable products.

My technical background includes React, Redux, GraphQL, Relay, Formik, Node.js, Express.js, PostgreSQL, MongoDB, Git, and modern development workflows. Alongside development, I've contributed to code reviews, prototype development, quality assurance testing, and documentation to support efficient team collaboration and continuous improvement.

With a foundation in both technology and the humanities, I bring a combination of analytical thinking, problem-solving, communication, and attention to detail to every project. I'm passionate about building software that is not only functional and scalable but also meaningful and easy to use.
